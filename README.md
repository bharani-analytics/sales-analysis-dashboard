# sales-analysis-dashboard
SQL and Powerbi project analyzing sales data


##  Project Overview
This project focuses on analyzing sales data to uncover key business insights. Using SQL for data cleaning and Power BI for visualization, the dashboard helps understand sales performance across regions, categories, and time.
## Tools Used
- SQL (MySQL)
- Power BI
- Excel

##  Key Insights
- Total Sales, Orders, and Customers overview
- Region-wise sales performance
- Category & Sub-category analysis
- Monthly sales trend analysis
- Identification of top-performing segments

## Dashboard Features

- Interactive filters (Year, Region, Category)
- Dynamic KPIs
- Clean and user-friendly layout
